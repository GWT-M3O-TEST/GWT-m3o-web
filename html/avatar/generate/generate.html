Generate an unique avatar
encode format of avatar image: `png` or `jpeg`; default is `jpeg`
avatar's gender: `male` or `female`; default is `male`
set to true to upload to the M3O CDN and receive the url
avatar's username, unique username will generate the unique avatar; if empty, every request generates a random avatar; if upload == true, username will be the CDN filename rather than a random uuid string
